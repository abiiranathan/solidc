# solidc

A collection of data structures, algorithms and advanced utilities in the
C programming language.

- Defer
- Logging
- OS and File operations
- B+ Tree
- Linked List
- Map
- Priority Queue
- Set
- Sorting
- Custom String and all string operations
- Vector
- Arena allocator
- Parsing command-line args in go-style flag package.
- Reading input from stdin the right way
- Network server and client abstractions
- Pub-Sub.
- Threadpool
