Detailed Description

Arena control structure.

The two hottest fields — curr and end — are placed first so they share a cache line with each other and with current_block, regardless of where the struct is allocated. aligned_alloc_xp(64, sizeof(Arena)) in arena_create() ensures the heap-allocated instance starts on a 64-byte boundary, keeping the fast path in a single cache line fetch.
